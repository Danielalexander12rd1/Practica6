¿Qué es FRONTED?


El frontend es la parte de una aplicación o sitio web con la que interactúan directamente los usuarios. Se refiere a todo lo que ves y con lo que interactúas en una página web, como botones, menús, formularios, texto, imágenes, y animaciones. El frontend se encarga de la interfaz de usuario (UI) y de la experiencia de usuario (UX).el frontend es la parte de una aplicación web o móvil que interactúa directamente con el usuario. Esta es la interfaz visible y con la que los usuarios interactúan, que incluye el diseño, la estructura, los botones, formularios, menús, imágenes y cualquier elemento que el usuario vea y use.

Objetivos principales del Frontned


  • Crear interfaces de usuario atractivas y funcionales.

  • Implementar interactividad y dinámica en la página web.

  • Optimizar el rendimiento de la aplicación para una carga rápida.

  • Asegurar que el sitio web sea responsive y accesible en todos los dispositivos.

  • Garantizar una experiencia de usuario fluida e intuitiva.

    ¿Cómo funciona el frontend?


    Renderizado en el navegador:

    Cuando visitas un sitio web, el navegador recibe el HTML, CSS, y JavaScript del servidor. El navegador procesa este código y construye la interfaz visible para el usuario.

    Interacción al usuario:
    Los usuarios interactúan con botones, formularios, y otros elementos de la interfaz. JavaScript se utiliza para captar estas acciones y responder sin necesidad de recargar la página (por ejemplo, mostrar un menú al hacer clic en un ícono).

    Comunicación con el backend:
    El frontend puede enviar y recibir datos del backend (el servidor) mediante solicitudes HTTP (usando AJAX o tecnologías como Fetch API o Axios). Esto permite actualizar el contenido sin recargar toda la página, proporcionando una mejor experiencia de usuario.
